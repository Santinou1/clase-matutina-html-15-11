¿ Que es HTML (HyperText Markup Language) ?

HTML es un lenguaje de Marcado / Estructura.

Cual es el atajo del teclado para realizar la estrucutra basica del HTML

El atajo del teclado es "!" y apretamos enter

¿ Que es una etiqueta ?

Una etiqueta es un componente fundamental que se usa para estructurar y definir el contenido que va a tener la pagina web.

¿ Como se construye une etiqueta ?

Las etiquetas estan construidas por un nombre reservado como puede ser "p" que hace referencia a parrafo o un "h1" que hace referencia a un titulo.

Y la palabra reservada se envuelve en simbolos de mayor y menor.

¿ Como se abre y cierra una etiqueta ?

Una etiqueta tiene apertura y cierre , para abrir la etiqueta se hace con "< palabra reservada> ", y para cerrar la etiqueta es con la barra diagonal despues del simbolo de la boca menor "< / palabra reservada>".

¿ Cuando determinamos que etiqueta usar ?

Las etiquetas la determinamos usar debido al comportamiento que querramos que tenga nuestra web. Si queremos marcar el titulo de nuestra pagina vamos a usar un "H1" , ahora si queremos realizar subtitulos junto a pequeños parrafos para explicar lo que dice el subtitulo , podemos usar la combinacion de "h2" + un parrafo "p".

Diferencia entre Article / Section y Div

Los 3 son contenedores, eso es lo fundamental , las 3 etiquetas van a contener diferente tipo de informacion.

No se usan todos los mismos contenedores aunque visualmente sean iguales debido al SEO.

Un buen uso de las etiquetas mejora el SEO.

Por ejemplo.

El proposito del "div" es ser un contenedor generico sin semantica propia. Y sus usos comunes vendrian a ser agrupar elementos para estilos o manipulacion sin importar el texto

El proposito del "section" es agrupar contenido relacionado tematicamente. Y sus usos comunes son dividir la pagina en secciones con un tema comun. Por ejemplo = Noticias, productos, etc

El proposito del "article" es ser un contenedor con contenido autonomo o independiente, Sus usos comunes podrian ser Articulos o piezas de contenido que pueden ser distribuidos o reutilizados, por ejemplo Blogs,noticias etc.
