Guia basica de HTML

Introduccion

HTML (HyperText Markup Language) es el lenguaje estandar para hacer paginas webs.

En esta clase vamos a aprenedr el uso de las etiquetas basicas como:

Heading <h1> : Para titulos y subtitulos.

Parrafo <p> : Para parrafos.

Div <div> : Para agrupar contenidos.

Ejercicio

Vamos a realizar un ejercicio de practica en el cual cuenten lo que ustedes quieran siguiendo las instrucciones de abajo.

1- Crea un Titulo Principal usando una etiqueta <h1>

2- Agrega al menos dos subtitulos utilizando <h2> o <h3>

3- Escribe un nuevo parrafo para cada subtitulo utilizando la etiqueta <p>

4- Envuelve cada seccion en un contenedor utilizando la etiqueta <div>
